# دليل تثبيت بوت واتساب (Manus) على خادم خارجي 🚀

هذا الدليل سيساعدك على تشغيل البوت الخاص بك بشكل دائم ومستقر (24/7) بعيداً عن بيئة العمل المؤقتة.

### 1. المتطلبات الأساسية
*   **خادم (VPS):** يمكنك استخدام (DigitalOcean, AWS, Google Cloud) أو أي خادم يدعم Python.
*   **اسم نطاق (Domain Name) مع شهادة SSL:** (ضروري لأن واتساب يتطلب رابط `https`).
*   **Python 3.10+** مثبت على الخادم.

### 2. خطوات التثبيت (على الخادم)
1.  **رفع الملفات:** قم برفع ملف `whatsapp_bot.py` و `requirements.txt` إلى مجلد في خادمك.
2.  **تثبيت المكتبات:**
    ```bash
    pip3 install -r requirements.txt
    ```
3.  **تشغيل البوت (بشكل دائم):**
    يفضل استخدام `gunicorn` لضمان استقرار البوت:
    ```bash
    gunicorn --bind 0.0.0.0:8080 whatsapp_bot:app --daemon
    ```

### 3. تحديث رابط الـ Webhook في Meta
بمجرد تشغيل البوت على خادمك، ستحصل على رابط جديد (مثلاً: `https://your-domain.com/webhook`).
1.  اذهب إلى [Meta for Developers](https://developers.facebook.com/).
2.  في إعدادات **WhatsApp > Configuration**.
3.  قم بتحديث **Callback URL** بالرابط الجديد الخاص بك.
4.  استخدم نفس الـ **Verify Token** (`ManusBot2026`).

### 4. ملاحظة هامة حول "الذاكرة"
النسخة الحالية تعمل بشكل ممتاز في معالجة الملفات والرد الآلي. إذا أردت في المستقبل إضافة "ذاكرة مستديمة" (قاعدة بيانات) لحفظ الطلبات السابقة، ستحتاج لربطها بقاعدة بيانات مثل SQLite أو PostgreSQL على نفس الخادم.

**بالتوفيق في مشروعك!** 🌟
